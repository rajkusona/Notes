## ?? User Workflows

### Creating a Note
```
1. Click "Create Note" in navbar
2. Fill in Title (required)
3. Write Content (required)
4. Select Priority (Low/Medium/High)
5. Click "Create Note"
6. See your note appear in grid!
```

### Editing a Note
```
1. Find note in grid
2. Click "Edit" button
3. Modify any field
4. Click "Save Changes"
5. See updated note instantly
```

### Deleting a Note
```
1. Find note in grid
2. Click "Delete" button
3. Review note on confirmation page
4. Click "Yes, Delete" to confirm
5. Note is removed
```
